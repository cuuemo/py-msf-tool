# py-msf-tool
* 需在Linux运行此py代码，Linux需要安装nmap和msfconsole不是特殊版本是自带python3,也可在kali直接运行

## 作用
* 这是一个通过python使用nmap和msfconsole自动渗透windows7或者windows server 2008操作系统（只能渗透未打补丁的系统）
* 目前仅支持Linux系统
* 渗透进去后可以使用sessions进行批量运行命令
* 将来可能会添加更多功能
